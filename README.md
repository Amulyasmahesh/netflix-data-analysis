# Netflix Content Analysis using Python

## Project Overview
This project analyzes Netflix content data to identify trends in movies and TV shows available on the platform.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Netflix Movies and TV Shows dataset containing information about titles, genres, countries, release years, and durations.

## Key Analysis Performed
- Movies vs TV Shows distribution
- Top countries producing Netflix content
- Content growth over years
- Most popular genres on Netflix
- Longest movies available

## Key Insights
- Netflix hosts significantly more movies than TV shows.
- The United States produces the highest number of titles.
- Content growth increased rapidly after 2016.
- International Movies and Dramas dominate the platform.

## Visualizations
The project includes charts showing:
- Movies vs TV Shows
- Content growth over time
- Top 10 producing countries
- Top genres
